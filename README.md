# Backend Zencode


Este é o backend da API de candidatos do teste ZENCODE.  
Aqui vai uma explicação simples de como usar o projeto.

---

## Requisitos para poder subir o projeto

- Node.js v24+ instalado
- NPM instalado
- Git instalado

---

## Clonar o repositório

Abre o terminal e escreve:

```bash
git clone https://github.com/jobmaria7/zencode-backend.git
cd zencode-backend
